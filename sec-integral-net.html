    Section 4.5 Net Change Theorem and Indefinite Integrals

    Objectives

    • Interpret a definite integral as net change of a quantity whose rate of change is known.
      🔗
    • Use indefinite integral notation to represent the family of all antiderivatives.
      🔗
    • Compute net change from a velocity (or rate) function using \(F(b)-F(a)\text{.}\)
      🔗
    • Compute the average value of a function on an interval.
      🔗
    • State and use the Mean Value Theorem for Integrals.
      🔗
    🔗

    Subsection Review: Four Meanings of the Definite Integral

    The definite integral \(\displaystyle \int_a^b f(x)\,dx\) has several complementary interpretations.
    🔗
    • Physical: Suppose \(y\) and \(z\) are physical quantities that depend continuously on an independent variable \(x\text{,}\) with \(y=f(x)\) and \(z=F(x)\text{.}\) If \(y\) is the rate of change of \(z\text{,}\) meaning \(y=\dfrac{dz}{dx}\text{,}\) then the integral of \(y\) over \([a,b]\) is the total change in \(z\) from \(x=a\) to \(x=b\text{.}\)
      🔗
      In Leibniz notation, this is written
      🔗
      \begin{equation*} \int_a^b y\,dx \ =\ \left.z\right|_{x=a}^{x=b}. \end{equation*}
      In Newton notation, if \(f(x)=F'(x)\text{,}\) then
      🔗
      \begin{equation*} \int_a^b f(x)\,dx \ =\ F(b)-F(a), \end{equation*}
      which is the Second Fundamental Theorem of Calculus. If an initial value \(F(a)\) is known, then
      🔗
      \begin{equation*} F(x)\ =\ F(a) + \int_a^x f(t)\,dt \qquad\text{and}\qquad F'(x)\ =\ \frac{d}{dx}\left(\int_a^x f(t)\,dt\right)\ =\ f(x), \end{equation*}
      which is the First Fundamental Theorem of Calculus.
      🔗
      🔗
    • Geometric: The integral is the signed area between the graph \(y=f(x)\) and the \(x\)-axis on \([a,b]\text{,}\) counting regions above the axis as positive and regions below as negative.
      🔗
      🔗
    • Numerical: Divide \([a,b]\) into \(n\) increments of width \(\Delta x=\dfrac{b-a}{n}\text{.}\) Choose sample points \(x_1,\ldots,x_n\) (one in each increment). Then the integral is approximated by a Riemann sum:
      🔗
      \begin{equation*} \int_a^b f(x)\,dx \ \approx\ \sum_{i=1}^n f(x_i)\,\Delta x \ =\ f(x_1)\Delta x + \cdots + f(x_n)\Delta x, \end{equation*}
      and the exact integral is the limit as \(n\to\infty\) (so \(\Delta x\to 0\)).
      🔗
      🔗
    • Algebraic: If we can find an antiderivative \(F\) with \(F'(x)=f(x)\text{,}\) then \(\displaystyle \int_a^b f(x)\,dx = F(b)-F(a)\text{.}\) Later we will develop systematic methods for finding antiderivatives (for example, substitution, which reverses the Chain Rule).
      🔗
      🔗
    🔗

    Subsection The Net Change Theorem

    Theorem 4.5.1. Net Change Theorem.

    If a quantity \(z\) changes with respect to \(x\) at a rate \(\dfrac{dz}{dx}=f(x)\text{,}\) then the net change in \(z\) on \([a,b]\) is
    🔗
    \begin{equation*} z(b)-z(a)\ =\ \int_a^b f(x)\,dx. \end{equation*}
    Equivalently, if \(F'(x)=f(x)\text{,}\) then \(z(b)-z(a)=F(b)-F(a)\text{.}\)
    🔗
    🔗
    The phrase net change emphasizes that increases and decreases both count: if the rate \(f(x)\) is sometimes positive and sometimes negative, the integral records the overall effect.
    🔗
    🔗

    Subsection Indefinite Integral Notation

    Because antiderivatives and integrals are linked by the Fundamental Theorems, we use the integral sign as notation for the most general antiderivative:
    🔗
    \begin{equation*} \int f(x)\,dx \ =\ F(x)+C, \qquad\text{where }F'(x)=f(x)\text{ and }C\text{ is any constant}. \end{equation*}
    The expression \(\displaystyle \int f(x)\,dx\) is called an indefinite integral. It represents a family of functions (all antiderivatives), not a single number.
    🔗

    Example 4.5.2.

    Compute \(\displaystyle \int x^2\,dx\text{.}\)
    🔗
    Solution.
    Since \(\dfrac{d}{dx}\left(\tfrac13 x^3\right)=x^2\text{,}\) we have
    🔗
    \begin{equation*} \int x^2\,dx \ =\ \tfrac13 x^3 + C. \end{equation*}
    🔗
    🔗

    Example 4.5.3.

    A car has velocity \(v(t)=10t+10\sin(\pi t)\) (meters per second). How far does it travel from \(t=0\) to \(t=3\text{?}\)
    🔗
    Solution.
    Distance traveled is net change in position: \(\displaystyle s(3)-s(0)=\int_0^3 v(t)\,dt\text{.}\) First find an antiderivative:
    🔗
    \begin{equation*} \int \left(10t+10\sin(\pi t)\right)\,dt \ =\ 5t^2 - \frac{10}{\pi}\cos(\pi t) + C. \end{equation*}
    Now evaluate as a definite integral:
    🔗
    \begin{equation*} \begin{array}{rcl} \displaystyle \int_0^3 \left(10t+10\sin(\pi t)\right)\,dt &=& \left. \left(5t^2 - \frac{10}{\pi}\cos(\pi t)\right)\right|_{t=0}^{t=3} \\[0.9em] &=& \left(5(3^2) - \frac{10}{\pi}\cos(3\pi)\right) - \left(5(0^2) - \frac{10}{\pi}\cos(0)\right) \\[0.9em] &=& \left(45 - \frac{10}{\pi}(-1)\right) - \left(0 - \frac{10}{\pi}(1)\right) \\[0.9em] &=& 45 + \frac{20}{\pi} \ \approx\ 51.4\ \text{meters.} \end{array} \end{equation*}
    🔗
    🔗
    🔗

    Subsection Average Value of a Function

    In a Riemann sum approximation, \(\sum_{i=1}^n f(x_i)\Delta x\) can be rewritten to highlight an average of the sample values:
    🔗
    \begin{equation*} \begin{array}{rcl} \displaystyle \sum_{i=1}^n f(x_i)\Delta x &=& \displaystyle \sum_{i=1}^n f(x_i)\,\frac{b-a}{n} \\[0.9em] &=& \displaystyle (b-a)\,\frac{f(x_1)+\cdots+f(x_n)}{n}. \end{array} \end{equation*}
    Taking the limit as \(n\to\infty\) motivates the definition of the average value of \(f\) on \([a,b]\text{:}\)
    🔗

    Definition 4.5.4.

    The average value of \(f\) on \([a,b]\) is
    🔗
    \begin{equation*} f_{\text{avg}} \ =\ \frac{1}{b-a}\int_a^b f(x)\,dx. \end{equation*}
    🔗

    Example 4.5.5.

    Find the average value of \(f(x)=\sqrt{x}\) on \([0,4]\text{.}\)
    🔗
    Solution.
    An antiderivative is \(\int x^{1/2}\,dx=\frac{2}{3}x^{3/2}+C\text{.}\) Therefore,
    🔗
    \begin{equation*} \begin{array}{rcl} \displaystyle f_{\text{avg}} &=& \displaystyle \frac{1}{4-0}\int_0^4 \sqrt{x}\,dx \\[0.9em] &=& \displaystyle \frac{1}{4}\left.\left(\frac{2}{3}x^{3/2}\right)\right|_{x=0}^{x=4} \\[0.9em] &=& \displaystyle \frac{1}{4}\cdot\frac{2}{3}\cdot 4^{3/2} \ =\ \frac{1}{4}\cdot\frac{2}{3}\cdot 8 \ =\ \frac{4}{3} \ \approx\ 1.3. \end{array} \end{equation*}
    Even though \(\sqrt{x}\) ranges from 0 to 2 on \([0,4]\text{,}\) the average is above 1 because the graph bulges upward.
    🔗
    🔗
    🔗

    Example 4.5.6.

    Let \(s(t)\) be a position function with velocity \(v(t)=s'(t)\text{.}\) Show that the average value of \(v\) on \([a,b]\) equals total distance traveled divided by elapsed time.
    🔗
    Solution.
    Using the definition of average value and the Second Fundamental Theorem:
    🔗
    \begin{equation*} \begin{array}{rcl} \displaystyle v_{\text{avg}} &=& \displaystyle \frac{1}{b-a}\int_a^b v(t)\,dt \\[0.9em] &=& \displaystyle \frac{1}{b-a}\int_a^b s'(t)\,dt \\[0.9em] &=& \displaystyle \frac{1}{b-a}\left(s(b)-s(a)\right) \ =\ \frac{s(b)-s(a)}{b-a}. \end{array} \end{equation*}
    🔗
    🔗
    🔗

    Subsection Mean Value Theorem for Integrals

    Theorem 4.5.7. Mean Value Theorem for Integrals.

    If \(f\) is continuous on \([a,b]\text{,}\) then there exists a point \(c\in(a,b)\) such that
    🔗
    \begin{equation*} f(c)\ =\ \frac{1}{b-a}\int_a^b f(x)\,dx. \end{equation*}
    In words: at some point, the function attains its average value.
    🔗
    🔗
    Proof idea. Let \(F(x)=\int_a^x f(t)\,dt\text{.}\) By the Mean Value Theorem for derivatives, there is \(c\in(a,b)\) such that \(F'(c)=\dfrac{F(b)-F(a)}{b-a}\text{.}\) By the First Fundamental Theorem, \(F'(c)=f(c)\text{.}\) Also \(F(b)-F(a)=\int_a^b f(x)\,dx\text{.}\) Substituting gives the result.
    🔗
    In the example \(f(x)=\sqrt{x}\) on \([0,4]\text{,}\) we found \(f_{\text{avg}}=\frac{4}{3}\text{.}\) So there is \(c\in(0,4)\) with \(\sqrt{c}=\frac{4}{3}\text{,}\) meaning \(c=\frac{16}{9}\text{.}\)
    🔗
    🔗
