    Section 4.1 Antiderivatives

    Objectives

    • Interpret an antiderivative as “reversing differentiation” and connect it to rates of change.
      🔗
    • Find general antiderivatives using basic antidifferentiation rules and combine these over addition, subtraction, or multiplying by a constant.
      🔗
    • Solve simple initial value problems by using a given function value to determine the unknown constant.
      🔗
    • Recognize when algebraic manipulation is needed before antidifferentiating (e.g., rewriting quotients or using identities).
      🔗
    • Reverse the Chain Rule in simple cases when a constant multiple is missing.
      🔗
    • Verify an antiderivative by differentiating and checking that it produces the original function.
      🔗
    • Interpret antiderivatives in applied contexts, including recovering velocity and position from acceleration.
      🔗
    🔗

    Subsection Reversing Differentiation

    In many applications we care about some unknown function \(F(x)\text{,}\) but what we can measure or model is its rate of change, the derivative \(F'(x)=f(x)\text{.}\) Finding \(F\) from \(f\) is the process of antidifferentiation. We call \(F\) an antiderivative (or primitive) of \(f\text{.}\) This is quite often spoken as “big \(F\) and little \(f\text{.}\)”
    🔗

    Example 4.1.1.

    Suppose \(F'(t)=t^2-1\text{.}\) Find a general formula for \(F(t)\text{.}\)
    🔗
    Solution.
    Since \((\tfrac13 t^3)'=t^2\) and \((-t)'=-1\text{,}\) so one antiderivative is \(F(t)=\tfrac13 t^3-t\text{.}\) However, this is not the only one. For example, \(G(t)=\tfrac13 t^3-t+5\) also satisfies \(G'(t)=t^2-1\text{.}\) Any constant disappears when differentiating, so the general antiderivative is
    🔗
    \begin{equation*} F(t)=\tfrac13 t^3-t+C. \end{equation*}
    Notice that this constant shifts the graph up or down but does not change slopes. Visually, the family of antiderivatives looks like a vertical stack of curves, all with the same shape and slopes, but different heights.
    🔗
    🔗
    🔗
    So as we saw in our previous example, there are infinitely many antiderivatives of a given function, which appear to be differing by a constant. However, if we also know a single value like \(F(0)=5\text{,}\) then we can determine \(C\) and get a unique function. Problems like this are called initial value problems.
    🔗

    Example 4.1.3.

    Find the unique function \(F\) satisfying \(F'(t)=t^2+2\) and \(F(0)=5\text{.}\)
    🔗
    Alternative Phrasing: Solve the initial value problem \(F'(t)=t^2+2\text{,}\) \(F(0)=5\text{.}\)
    🔗
    Solution.
    The general antiderivative of \(t^2+2\) is \(\tfrac13 t^3+2t+C\text{.}\) To find the particular solution, we use the initial condition \(F(0)=5\text{:}\)
    🔗
    \begin{equation*} F(0)=\tfrac13 (0)^3+2(0)+C=5 \Rightarrow C=5. \end{equation*}
    So the unique function is \(F(t)=\tfrac13 t^3+2t+5\text{.}\)
    🔗
    🔗
    🔗

    Definition 4.1.4. Antiderivative.

    A function \(F(x)\) is an antiderivative of \(f(x)\) on an interval if \(F'(x)=f(x)\) for all \(x\) in that interval.
    🔗
    🔗

    Theorem 4.1.5. Antiderivatives Differ by a Constant.

    Suppose \(F_0(x)\) is one antiderivative of \(f(x)\) on an interval. Then:
    🔗
    (a) The general antiderivative of \(f(x)\) is \(F(x)= F_0(x)+C\text{,}\) for any constant \(C\text{.}\) There are no other antiderivatives of \(f\) besides those of this form.
    🔗
    (b) For any point \(c\) in the interval and any real number \(A\text{,}\) there is exactly one antiderivative \(F(x)\) satisfying \(F'(x)=f(x)\) and \(F(c)=A\text{.}\)
    🔗
    🔗

    Proof.

    (a) If \(F\) is any antiderivative of \(f\text{,}\) then \(F'(x)=f(x)=F_0'(x)\text{.}\) By the Uniqueness Theorem from the Mean Value Theorem section, two functions with the same derivative differ by a constant. Therefore \(F(x)=F_0(x)+C\text{.}\)
    🔗
    (b) By part (a), any antiderivative has the form \(F(x)=F_0(x)+C\text{.}\) Requiring \(F(c)=A\) gives \(F_0(c)+C=A\text{,}\) so \(C=A-F_0(c)\text{,}\) which is a single value.
    🔗
    🔗
    🔗

    Subsection Basic Antidifferentiation Rules

    Antidifferentiation starts by reversing the derivative rules you already know. The key is to build a library of basic patterns, then we can extend these to more complicated functions using our rules of derivatives.
    🔗

    Theorem 4.1.6. Basic Antiderivative Rules.

    For each of the following let \(F(x)\) be an antiderivative of \(f(x)\text{.}\) Then:
    • If \(f(x)=x^n\) for some integer \(n\neq -1\text{,}\) then \(F(x)=\frac{1}{n+1}x^{n+1}+C\text{.}\) (This is the power rule for antidifferentiation.)
      🔗
      🔗
    • If \(f(x)=\frac{1}{x}\text{,}\) then \(F(x)=\ln|x|+C\text{.}\) (This is a special case of the power rule, but it is important enough to be called out separately.)
      🔗
      🔗
    • Trigonometric functions:
      • If \(f(x)=\cos(x)\text{,}\) then \(F(x)=\sin(x)+C\text{.}\)
        🔗
        🔗
      • If \(f(x)=\sin(x)\text{,}\) then \(F(x)=-\cos(x)+C\text{.}\)
        🔗
        🔗
      • If \(f(x)=\sec^2(x)\text{,}\) then \(F(x)=\tan(x)+C\text{.}\)
        🔗
        🔗
      • If \(f(x)=\sec(x)\tan(x)\text{,}\) then \(F(x)=\sec(x)+C\text{.}\)
        🔗
        🔗
      🔗
      🔗
    • Exponential functions:
      • If \(f(x)=e^x\text{,}\) then \(F(x)=e^x+C\text{.}\)
        🔗
        🔗
      • If \(f(x)=a^x\) for some constant \(a\gt 0\) (and \(a\neq 1\)), then \(F(x)=\frac{1}{\ln(a)}a^x+C\text{.}\)
        🔗
        🔗
      🔗
      🔗
    • Inverse trig functions:
      • If \(f(x)=\dfrac{1}{\sqrt{1-x^2}}\text{,}\) then \(F(x)=\arcsin(x)+C\text{.}\)
        🔗
        🔗
      • If \(f(x)=\dfrac{1}{1+x^2}\text{,}\) then \(F(x)=\arctan(x)+C\text{.}\)
        🔗
        🔗
      🔗
      🔗
    🔗
    🔗

    Proof.

    Each of these can be verified by differentiating the proposed antiderivative and confirming you get back the original function.
    🔗
    🔗

    Remark 4.1.7.

    Notice the derivative of \(\ln|x|\) is \(\frac{1}{x}\) which takes into account both positive and negative values of \(x\text{,}\) so the antiderivative of \(\frac{1}{x}\) must include an absolute value.
    🔗
    If the interval you are working with has only positive values of \(x\text{,}\) then you can drop the absolute value and write the antiderivative as \(\ln(x)+C\text{.}\)
    🔗
    🔗
    Now that we have the building blocks for antiderivatives, we can also reverse the rules of derivatives to find antiderivatives of more complicated functions with sums / differences / constants.
    🔗

    Theorem 4.1.8. Sums and Constant Multiples.

    If \(F'(x)=f(x)\) and \(G'(x)=g(x)\text{,}\) then for constants \(a,b\text{,}\) an antiderivative of \(af(x)+bg(x)\) is \(aF(x)+bG(x)\text{.}\)
    🔗
    🔗

    Proof.

    This follows from the linearity of derivatives: \((aF(x)+bG(x))' = aF'(x)+bG'(x)=af(x)+bg(x)\text{.}\)
    🔗
    🔗
    Now let’s see some examples!
    🔗
    🔗

    Subsection Select Examples

    Example 4.1.9.

    Find a general antiderivative of \(f(x)=7x^3-x\sqrt{x}+\dfrac{3}{x^2}-\dfrac{4}{\cos^2(x)} + e^x\text{.}\)
    🔗
    Solution.
    First let’s rewrite our function as a sum of basic patterns: \(f(x)=7x^3-x^{3/2}+3x^{-2}-4\sec^2(x)+e^x\text{.}\) Then we can antidifferentiate term-by-term using the rules we know:
    🔗
    \begin{equation*} \begin{aligned} F(x) &= 7\cdot \frac{1}{4}x^4 \;-\; \frac{1}{\frac52}x^{5/2} \;+\; 3\cdot\frac{1}{-1}x^{-1} \;-\;4\tan(x) + e^x + C\\[0.3em] &= \frac{7}{4}x^4 \;-\; \frac{2}{5}x^{5/2} \;-\;\frac{3}{x}\;-\;4\tan(x)+e^x+C. \end{aligned} \end{equation*}
    A quick check is to differentiate \(F(x)\) and confirm you get back \(f(x)\text{.}\)
    🔗
    🔗
    🔗
    We can also reverse the Chain Rule when we see a composition, but the constant multiple is missing.
    🔗

    Example 4.1.10.

    Find the antiderivative of \(f(x)=\cos(3x)\) such that \(F(0)=2\text{.}\)
    🔗
    Solution.
    Since \((\sin(3x))'=3\cos(3x)\text{,}\) we divide by 3 to get the general antiderivative of \(\cos(3x)\text{:}\)
    🔗
    \begin{equation*} F(x)=\tfrac13\sin(3x)+C. \end{equation*}
    Now use the initial condition \(F(0)=2\text{:}\) \(\tfrac13\sin(0)+C=2\text{,}\) so \(C=2\text{.}\) Therefore
    🔗
    \begin{equation*} F(x)=\tfrac13\sin(3x)+2. \end{equation*}
    Again, we can check by differentiating \(F(x)\) and confirming we get back \(f(x)\text{.}\)
    🔗
    🔗
    🔗
    When the function is a product or quotient, we cannot simply antidifferentiate each factor and hope to get an antiderivative of the whole function. In general, there is no simple formula for the antiderivative of a product or quotient. However, if we can rewrite the expression as a sum of basic patterns, then we can antidifferentiate term-by-term.
    🔗

    Example 4.1.11.

    Find a general antiderivative of \(\displaystyle f(x)=\frac{\sqrt{x}+4}{x}\text{.}\)
    🔗
    Solution.
    Rewrite as a sum of powers: \(\displaystyle \frac{\sqrt{x}+4}{x}=\frac{x^{1/2}}{x}+\frac{4}{x}=x^{-1/2}+4x^{-1}\text{.}\) Then antidifferentiate term-by-term:
    🔗
    \begin{equation*} \begin{aligned} F(x) &= \frac{1}{\frac12}x^{1/2}+4\cdot\ln|x|+C\\ &= 2x^{1/2}+4\ln|x|+C\\ &= 2\sqrt{x}+4\ln|x|+C. \end{aligned} \end{equation*}
    🔗
    🔗

    Remark 4.1.12. Products and Quotients.

    Be careful with products and quotients! In general, the antiderivative of a product is not the product of antiderivatives (same for quotients). We will learn how to handle these later, but for now, we can sometimes antidifferentiate products or quotients if we can expand them into sums of Basic Antiderivatives.
    🔗
    🔗

    Example 4.1.13.

    Find a general antiderivative of \(f(x)=\sin^2(x)\text{.}\)
    🔗
    Hint.
    Use the identity \(\cos(2x)=1-2\sin^2(x)\) to rewrite \(\sin^2(x)\) as a sum of basic patterns.
    🔗
    🔗
    Solution.
    The function \(\sin^2(x)\) is not on our basic list (it is the product of two sines) so we must be careful. Using the double angle formula from trig we know \(\cos(2x)=1-2\sin^2(x)\text{,}\) which gives
    \begin{equation*} \sin^2(x)=\tfrac12(1-\cos(2x)) = \tfrac12 - \tfrac12\cos(2x) \end{equation*}
    Now we can find the antidifferentiate:
    🔗
    \begin{equation*} \begin{aligned} F(x) &= \tfrac12 x - \tfrac12\cdot \tfrac{1}{2}\sin(2x) + C\\ &= \tfrac12 x - \tfrac14\sin(2x) + C. \end{aligned} \end{equation*}
    We can check by differentiating \(F(x)\) and confirming we get back \(f(x)\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Subsection Application: Acceleration, Velocity, and Position

    Previously we have seen how derivatives connect position, velocity, and acceleration in motion problems. Now we can use antidifferentiation to go in the opposite direction. If we know acceleration and some initial conditions, we can recover velocity and position by antidifferentiating twice.
    🔗

    Example 4.1.14.

    A rocket has acceleration \(a(t)=t+1\) (in \(\text{m}/\text{sec}^2\)). At time \(t=1\text{,}\) its height is \(s(1)=10\) meters and its velocity is \(v(1)=3\) m/sec. Find the velocity function \(v(t)\) and the height function \(s(t)\text{.}\)
    🔗
    Solution.
    Since \(v'(t)=a(t)=t+1\text{,}\) antidifferentiate:
    🔗
    \begin{equation*} v(t)=\tfrac12 t^2+t+C. \end{equation*}
    Use \(v(1)=3\text{:}\) \(\tfrac12(1)^2+1+C=3\text{,}\) so \(C=\tfrac32\text{.}\) Thus
    🔗
    \begin{equation*} v(t)=\tfrac12 t^2+t+\tfrac32. \end{equation*}
    Now antidifferentiate again because \(s'(t)=v(t)\text{:}\)
    🔗
    \begin{equation*} \begin{aligned} s(t) &= \tfrac12\cdot\tfrac13 t^3+\tfrac12 t^2+\tfrac32 t + B\\ &= \tfrac16 t^3+\tfrac12 t^2+\tfrac32 t + B. \end{aligned} \end{equation*}
    Use \(s(1)=10\text{:}\) \(\tfrac16+\tfrac12+\tfrac32+B=10\text{,}\) so \(B=\tfrac{47}{6}\text{.}\) Therefore
    🔗
    \begin{equation*} s(t)=\tfrac16 t^3+\tfrac12 t^2+\tfrac32 t+\tfrac{47}{6}. \end{equation*}
    With both conditions applied, the constants are determined and the solution is unique.
    🔗
    🔗

    Exercises Practice / Study Problems

    Some True/False Problems.

    Determine whether each statement is true or false and briefly justify your reasoning.
    🔗
    1.
    If \(F'(x)=f(x)\) and \(G'(x)=f(x)\) on an interval, then \(F(x)-G(x)\) is constant on that interval.
    🔗
    Solution.
    True. Since \(F'(x)=G'(x)\text{,}\) the function \(F(x)-G(x)\) has derivative zero, so it is constant.
    🔗
    🔗
    🔗
    2.
    \(\dfrac{d}{dx}\bigl(x^5+3\bigr)=5x^4\text{,}\) so one antiderivative of \(5x^4\) is \(x^5+3\text{.}\)
    🔗
    Solution.
    True. The derivative of \(x^5+3\) is indeed \(5x^4\text{,}\) so it is an antiderivative. In fact, the general antiderivative is \(x^5+C\text{.}\)
    🔗
    🔗
    🔗
    3.
    If \(f(x)=\cos(x)\text{,}\) then a general antiderivative is \(\sin(x)+C\text{.}\)
    🔗
    Solution.
    True. The derivative of \(\sin(x)+C\) is \(\cos(x)\text{,}\) so it is the general antiderivative of \(\cos(x)\text{.}\)
    🔗
    🔗
    🔗
    4.
    An antiderivative of \(\sin(x)\) is \(\cos(x)\text{.}\)
    🔗
    Solution.
    False. The derivative of \(\cos(x)\) is \(-\sin(x)\text{,}\) so it is not an antiderivative of \(\sin(x)\text{.}\) In fact, the general antiderivative of \(\sin(x)\) is \(-\cos(x)+C\text{.}\)

    6.
    If \(F'(x)=e^x\) and \(F(0)=7\text{,}\) then \(F(x)=e^x+7\text{.}\)
    🔗
    Solution.
    False. The antiderivative of \(e^x\) is \(e^x+C\text{.}\) Applying the initial condition \(F(0)=7\text{,}\) we get:
    \begin{equation*} F(0)=e^0+C=1+C=7 \Rightarrow C=6. \end{equation*}
    So the correct function is \(F(x)=e^x+6\text{.}\)

    8.
    The antiderivative of a sum equals the sum of antiderivatives.
    🔗
    Solution.
    True. If \(F'(x)=f(x)\) and \(G'(x)=g(x)\text{,}\) then \((F(x)+G(x))' = F'(x)+G'(x)=f(x)+g(x)\text{.}\) So an antiderivative of \(f(x)+g(x)\) is \(F(x)+G(x)\text{.}\)

    9.
    The antiderivative of a product equals the product of antiderivatives.
    🔗
    Solution.
    False. In general, the antiderivative of a product is not the product of antiderivatives. For example, if \(f(x)=x\) and \(g(x)=x^2\text{,}\) then an antiderivative of \(f(x)g(x)=x^3\) is \(\frac{1}{4}x^4+C\text{,}\) but the product of antiderivatives is \(\bigl(\frac{1}{2}x^2+C_1\bigr)\bigl(\frac{1}{3}x^3+C_2\bigr)\text{,}\) which is not an antiderivative of \(x^3\text{.}\)

    13.
    If \(f(x)=\dfrac{\cos(x)}{x}\text{,}\) then a general antiderivative is \(\dfrac{\sin(x)}{\tfrac12 x^2} +C\text{.}\)
    🔗
    Solution.
    False. According to the quotient rule for derivatives, the derivative of \(\dfrac{\sin(x)}{\tfrac12 x^2} +C\) is
    \begin{equation*} \dfrac{\cos(x)\cdot\tfrac12 x^2 - \sin(x)\cdot x}{\tfrac14 x^4} = \dfrac{\cos(x) - 2\sin(x)/x}{x^2} \end{equation*}
    which is not equal to \(\dfrac{\cos(x)}{x}\text{.}\) In fact, the antiderivative of \(\dfrac{\cos(x)}{x}\) cannot be expressed in terms of elementary functions. Again, there is no simple formula for the antiderivative of a quotient in general.
